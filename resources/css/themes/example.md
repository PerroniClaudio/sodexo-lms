# Tema Example

## Palette Colori

Questo è un tema di esempio che mostra come configurare un tema personalizzato per un nuovo cliente.

### Colori Principali
- **Primary:** `#16A34A` - Verde brillante
- **Secondary:** `#10B981` - Verde smeraldo
- **Accent:** `#8B5CF6` - Viola per contrasto

### Colori Base
- **Base 100:** `#FFFFFF` - Sfondo bianco puro
- **Base 200:** `#F0FDF4` - Sfondo verde molto chiaro
- **Base 300:** `#DCFCE7` - Sfondo verde chiaro
- **Base Content:** `#14532D` - Testo verde scuro

### Colori di Stato
- **Info:** Blu informativo
- **Success:** Verde successo
- **Warning:** Giallo/arancione avvisi
- **Error:** Rosso errori

### Stile
- **Border Radius:** 0.75rem (più squadrato)
- **Border Width:** 2px (più marcato)
- **Color Scheme:** Light

## Uso

Per usare questo tema in un deploy:

```env
APP_NAME="Example Company LMS"
APP_THEME=example
APP_URL=https://lms.example.com
```

## Note
- Tema dimostrativo per nuovi clienti
- Modifica i colori secondo il brand del cliente
- Mantieni coerenza tra primary/secondary/accent per una buona UX
