# Controlled MP4 Download with Mux

## Requirements

- mp4_support enabled on asset creation
- Public or signed playback ID

## Steps

1. Create asset with mp4_support
2. Store playback_id
3. Generate MP4 URL from playback ID
4. Protect access (signed URLs recommended)

## Notes

- MP4s are derived files, not originals
- Bandwidth costs apply
- Prefer signed URLs for private content
