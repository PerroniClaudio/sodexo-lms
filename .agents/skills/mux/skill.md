# Mux Video API – Laravel Integration Skill

This skill enables an agent to assist with integrating the Mux Video API

into Laravel applications, following only officially supported workflows.

Primary use cases:

- Video uploads (Direct Uploads)
- Asset lifecycle management
- Streaming via HLS
- Controlled MP4 downloads
- Playback security
- Webhook handling

This skill is model-agnostic and can be used with GPT‑4.1, GPT‑4.1‑mini,

or other LLMs.
